# projet_sig
Plateforme d’analyse géospatiale basée sur Python et JupyterLab pour le traitement des données satellitaires (NDVI, LULC, inondations, carbone) au Burkina Faso.
🌍 Projet SIG : Analyse Géospatiale et Télédétection Appliquée au Burkina Faso

🧭 Contexte

Dans un contexte marqué par les changements climatiques et la pression anthropique sur les ressources naturelles, l’analyse géospatiale constitue un outil essentiel pour le suivi environnemental et l’aide à la décision. Ce projet s’inscrit dans une démarche scientifique visant à exploiter les données satellitaires pour caractériser les dynamiques spatio-temporelles au Burkina Faso.

🎯 Objectif général

Développer une plateforme d’analyse géospatiale basée sur JupyterLab permettant le traitement, l’analyse et la visualisation de données satellitaires multi-sources.

🎯 Objectifs spécifiques
Cartographier l’occupation du sol (LULC)
Calculer des indices de végétation (NDVI, EVI)
Détecter et analyser les zones inondées
Estimer les dynamiques de carbone
Produire des indicateurs environnementaux fiables
🧪 Méthodologie

Le projet repose sur une approche intégrée combinant :

Traitement d’images satellitaires (Sentinel-1, Landsat, MODIS)
Analyse temporelle multi-dates
Calcul d’indices spectraux
Classification et détection de changements
Analyse spatiale sous Python
🧰 Environnement technique
Python (xarray, rasterio, numpy, matplotlib)
SIG et télédétection
Plateforme de versionnement : GitHub
