# Projet SIG – Analyse Environnementale

## Modules
- NDVI (végétation)
- Flood (inondations)
- LULC (occupation du sol)
- Carbone_PONASI (stock de carbone)

## Structure
Chaque module contient :
- data/raw : données brutes
- data/processed : données traitées
- notebooks : notebooks Jupyter
- scripts : scripts Python
- outputs/maps, tables, graphs : résultats

## Objectifs
- Suivi dynamique de la végétation
- Cartographie des inondations
- Analyse de l'occupation du sol
- Estimation des stocks de carbone
